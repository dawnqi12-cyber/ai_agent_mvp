# Backtest

Simplified strategy simulation for research prototypes.

Implemented strategy simulations:

- Covered Call
- Protective Put
- Bull Call Spread

The simulator uses historical close prices, estimated option premiums, fixed
holding windows, and expiration payoff. It outputs cumulative return, max
drawdown, win rate, average return, an equity curve, and per-trade details.

Important: this is for education and research only. It ignores many real-world
trading details and is not financial advice.
